---
title: "Supported AWS Regions"
---

Convox will run in any region that supports EC2 Container Service, Lambda and at least two Availability Zones.

## Currently supported regions:

- ap-northeast-1
- ap-northeast-2
- ap-southeast-1
- ap-southeast-2
- ca-central-1
- eu-central-1
- eu-west-1
- eu-west-2
- us-east-1
- us-east-2
- us-west-1
- us-west-2
