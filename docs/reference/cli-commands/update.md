# update

Update the Convox CLI to the latest available version. If a specific version is provided, the CLI is updated to that version instead.

## Syntax

```bash
$ convox update [version]
```

## Flags

| Flag | Short | Description |
|:-----|:------|:------------|
| `--rack` | `-r` | Rack name |

## Example Usage

```bash
$ convox update
Updating to 3.18.5... OK
```
